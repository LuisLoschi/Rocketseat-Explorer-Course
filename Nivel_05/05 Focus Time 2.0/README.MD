## 💻 Project
The Focus Timer Project can be used to manage your time in any type of activity!

### Features:
- Light/dark mode
- Reset clock
- Stop sound when time is up
- Play/stop 4 types of relaxing music
- Button to add or decrease 5 minutes on the timer
